📖 Overview

This project demonstrates various computer vision techniques applied to real-time video capture. The application provides an interactive GUI where users can select different image processing operations and adjust parameters dynamically.

Author: Diego André Sant'Ana (diego.santana@ifms.br)

✨ Key Features

🎥

Real-time Capture

Live webcam video processing with instant feedback

🖼️

Side-by-Side View

Compare original and processed images simultaneously

⚙️

12 Operations

Multiple filtering, edge detection, and morphological operations

🎛️

Adjustable Params

Fine-tune kernel sizes and threshold values in real-time

📋 Available Operations

🔍 Filtering

Adaptive Threshold (Gaussian)

Adaptive thresholding with Gaussian weighting for better local contrast

Adaptive Threshold (Mean)

Adaptive thresholding using mean of neighborhood area

📐 Edge Detection

Find Contours

Detect and draw object boundaries - adjust Min/Max threshold values

Canny Edge Detection

Classic multi-stage edge detection algorithm

🔷 Morphological Operations

Black Hat

Difference between closing operation and original image

Erode

Shrinks bright regions, removes small white noise

Dilate

Expands bright regions, fills small holes

Open

Erosion followed by dilation - removes small objects

Close

Dilation followed by erosion - fills small holes

Gradient

Difference between dilation and erosion - highlights edges

✂️ Segmentation

Threshold Binary Inverse

Binary thresholding with inverted output

Threshold Binary

Standard binary thresholding for image segmentation

🚀 How to Use

  1. 1
    Clone the repository

    Download or clone this project to your local machine

  2. 2
    Install requirements

    Install all Python packages listed above

  3. 3
    Run the application python pyAplicativo.py
  4. 4
    Activate camera

    Click "Active Capture" button to start webcam

  5. 5
    Select operation

    Choose from 12 different computer vision operations

  6. 6
    Adjust parameters

    Use spinboxes to fine-tune Kernel, Min, and Max values
